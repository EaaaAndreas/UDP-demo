Denne repo forsøger at demonstrere, hvordan man kan porte til sin RP2, med UDP.

I `/src` finder du forskellige demoer, der demonstrerer, hvordan man kan bruge `socket` med UDP.
Der findes også et modul, `UDP_sender`, som du kan kopiere over i dit eget projekt, og bruge til at lave en UDP-listener.
Du kan se et eksempel på brug, i bunden af scriptet.

# Download
Download demoen via terminalen.
```bash
git clone https://github.com/EaaaAndreas/UDP-demo ./UDP-demo
```
